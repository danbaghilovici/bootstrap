Usage Instructions

To use this Trophy-inspired Bootstrap theme in your project:

  1. Include the compiled bootstrap-trophy.css file in your HTML
  2. Add Font Awesome for icons: <link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css" rel="stylesheet">
  3. Include the Open Sans font: <link href="https://fonts.googleapis.com/css2?family=Open+Sans:wght@400;600;700&display=swap" rel="stylesheet">
  4. Use the button classes as shown in the examples above

Available Button Classes:

  • Styles: .btn-trophy-minimal, .btn-trophy-outline, .btn-trophy-solid
  • Colors: Add -secondary for the turquoise accent color
  • Sizes: .btn-trophy-xs, .btn-trophy-sm, .btn-trophy-md, .btn-trophy-lg, .btn-trophy-xl

Example HTML:

<a href="#" class="btn btn-trophy-solid-secondary btn-trophy-lg">
    Get Started <i class="fas fa-chevron-right"></i>
</a>
