HTML Structure Reference

Copy this HTML structure to use the switch component.

<!-- Default switch -->
<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" role="switch" id="switchId">
  <label class="form-check-label" for="switchId">Switch label</label>
</div>

<!-- Small variant -->
<div class="form-check form-switch form-switch-sm">
  <input class="form-check-input" type="checkbox" role="switch" id="switchSmall">
  <label class="form-check-label" for="switchSmall">Small switch</label>
</div>

<!-- Checked (ON) state -->
<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" role="switch" id="switchChecked" checked>
  <label class="form-check-label" for="switchChecked">Checked switch</label>
</div>

<!-- Disabled state -->
<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" role="switch" id="switchDisabled" disabled>
  <label class="form-check-label" for="switchDisabled">Disabled switch</label>
</div>

CSS Classes Reference

Class Description
.form-check Base wrapper class (Bootstrap)
.form-switch Switch modifier on wrapper
.form-switch-sm Small variant modifier
.form-check-input The actual checkbox input (styled as switch)
.form-check-label Label element for the switch
