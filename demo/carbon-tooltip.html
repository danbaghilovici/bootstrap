Carbon Tooltip Demo

Tooltip component styled with Carbon Design System specifications.

Tooltip Placements

Tooltips can be positioned on top, bottom, left, or right of the trigger element.

Tooltip on Links

Tooltips can be attached to inline links for contextual help.

Hover over this linked text to see the tooltip.

Long Text Tooltip

Tooltips can contain longer descriptions. Max-width is 288px.

Focus Trigger

Tooltips can be triggered by keyboard focus for accessibility.

HTML Structure Reference

Copy this HTML structure to use the tooltip component.

<!-- Button with tooltip -->
<button type="button" class="btn btn-secondary"
        data-bs-toggle="tooltip"
        data-bs-placement="top"
        title="Tooltip text">
  Button text
</button>

<!-- Link with tooltip -->
<a href="#" data-bs-toggle="tooltip" title="Tooltip text">
  Link text
</a>

<!-- Focus trigger (for inputs) -->
<input type="text" class="form-control"
       data-bs-toggle="tooltip"
       data-bs-trigger="focus"
       title="Tooltip text">

<!-- Initialize all tooltips via JavaScript -->
<script>
  const tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]');
  const tooltipList = [...tooltipTriggerList].map(el => new bootstrap.Tooltip(el));
</script>

CSS Classes Reference

Class Description
.tooltip Main tooltip container (generated by JS)
.tooltip-inner Inner content wrapper with background and padding
.tooltip-arrow Arrow/caret element pointing to trigger
.bs-tooltip-top Tooltip positioned above trigger
.bs-tooltip-bottom Tooltip positioned below trigger
.bs-tooltip-start Tooltip positioned to the left (start)
.bs-tooltip-end Tooltip positioned to the right (end)
