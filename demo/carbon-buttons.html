Disabled States

Disabled buttons are non-interactive with cursor: not-allowed

Note: Carbon specifies #c6c6c6 background with #8d8d8d text for disabled buttons. Bootstrap's variable system only supports opacity-based disabled styling. This is a documented limitation of the variable-only customization approach.
