Text Colors

Body Text (text-01) - #161616 (Gray 100)

This is primary body text using Carbon's text-01 token. It provides maximum contrast for readability.

Secondary Text (text-02) - #525252 (Gray 70)

This is secondary text using Carbon's text-02 token. Used for supporting information.

Tertiary Text (text-03) - #8d8d8d (Gray 50)

This is tertiary text using Carbon's text-03 token. Used for placeholder or hint text.

Links

Links use Carbon Blue 60 (#0f62fe) as the default color and Blue 70 (#0043ce) on hover. This is an example link - hover to see the color change.

Links can also appear in different contexts: Primary link, Secondary link, Success link, Danger link.
