HTML Structure Reference

Copy this HTML structure to use the progress bar component.

<!-- Default progress bar -->
<div class="progress" role="progressbar"
     aria-valuenow="50" aria-valuemin="0" aria-valuemax="100">
  <div class="progress-bar" style="width: 50%"></div>
</div>

<!-- Small variant -->
<div class="progress progress-sm" role="progressbar">
  <div class="progress-bar" style="width: 50%"></div>
</div>

<!-- Success state -->
<div class="progress progress-success" role="progressbar">
  <div class="progress-bar" style="width: 100%"></div>
</div>

<!-- Error state -->
<div class="progress progress-error" role="progressbar">
  <div class="progress-bar" style="width: 65%"></div>
</div>

<!-- With label -->
<div class="progress" role="progressbar" style="height: 1.25rem;">
  <div class="progress-bar" style="width: 75%">75%</div>
</div>

Carbon Design Specifications

Property Value Carbon Token
Track background #f4f4f4 $layer
Track height (default) 8px (0.5rem) Carbon standard
Track height (small) 4px (0.25rem) Carbon small variant
Border radius 0 Square corners
Box shadow none Flat design
Bar color (default) #0f62fe $interactive
Bar color (success) #24a148 $support-success
Bar color (error) #da1e28 $support-error
Text on bar #ffffff White
Min width 48px Carbon minimum

CSS Classes Reference

Class Description
.progress Main container (track), default 8px height
.progress-bar Inner bar/indicator element
.progress-sm Small variant modifier (4px height)
.progress-success Success state on container (green bar)
.progress-bar-success Success state on bar element
.progress-error Error state on container (red bar)
.progress-bar-error Error state on bar element
.progress-stacked Container for stacked progress bars
