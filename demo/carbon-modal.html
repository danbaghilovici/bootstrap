Carbon Modal Demo

Modal component styled with Carbon Design System specifications.

Default Modal

Standard modal with header, body, and footer. Uses default width (576px).

Modal Sizes

Modals are available in small (448px), large (672px), and extra-large (896px) sizes.

Scrollable Modal

Modal with scrollable body content when content exceeds viewport height.

Vertically Centered Modal

Modal centered vertically in the viewport.

Static Backdrop Modal

Modal that doesn't close when clicking outside. Requires explicit action to dismiss.

HTML Structure Reference

Copy this HTML structure to use the modal component.

<!-- Button trigger -->
<button type="button" class="btn btn-primary"
        data-bs-toggle="modal"
        data-bs-target="#exampleModal">
  Open Modal
</button>

<!-- Modal structure -->
<div class="modal fade" id="exampleModal" tabindex="-1"
     aria-labelledby="exampleModalLabel" aria-hidden="true">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title" id="exampleModalLabel">Modal Title</h5>
        <button type="button" class="btn-close"
                data-bs-dismiss="modal" aria-label="Close"></button>
      </div>
      <div class="modal-body">
        Modal body content here.
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary"
                data-bs-dismiss="modal">Cancel</button>
        <button type="button" class="btn btn-primary">Confirm</button>
      </div>
    </div>
  </div>
</div>

<!-- Size variants -->
<div class="modal-dialog modal-sm">...</div>  <!-- 448px -->
<div class="modal-dialog">...</div>           <!-- 576px default -->
<div class="modal-dialog modal-lg">...</div>  <!-- 672px -->
<div class="modal-dialog modal-xl">...</div>  <!-- 896px -->

<!-- Centered -->
<div class="modal-dialog modal-dialog-centered">...</div>

<!-- Scrollable body -->
<div class="modal-dialog modal-dialog-scrollable">...</div>

<!-- Static backdrop (add to modal element) -->
<div class="modal fade" data-bs-backdrop="static"
     data-bs-keyboard="false">...</div>

CSS Classes Reference

Class Description
.modal Main modal wrapper with backdrop
.modal-dialog Positions the modal content
.modal-content Container for header, body, footer
.modal-header Header with title and close button
.modal-title Modal heading (h5 recommended)
.modal-body Main content area
.modal-footer Action button container
.modal-sm Small modal (448px)
.modal-lg Large modal (672px)
.modal-xl Extra-large modal (896px)
.modal-dialog-centered Vertically center the modal
.modal-dialog-scrollable Scrollable modal body
Default Modal

This is a default modal with standard Carbon styling. Notice the square corners, subtle border, and drop shadow.

The modal uses 16px padding throughout for consistent spacing.

Small Modal

This is a small modal (448px width). Use for simple confirmations or short messages.

Large Modal

This is a large modal (672px width). Use for forms or content that needs more horizontal space.

The additional width allows for more complex layouts while maintaining readability.

Extra Large Modal

This is an extra-large modal (896px width). Use for complex content like data tables, multi-column forms, or detailed previews.

The maximum width provides ample space for intricate layouts and data-heavy interfaces.

Scrollable Modal

This modal demonstrates scrollable content. When the body content exceeds the viewport height, only the body scrolls while the header and footer remain fixed.

Centered Modal

This modal is vertically centered in the viewport. Use this variant when you want the modal to appear in the center of the screen.

Static Backdrop Modal

This modal won't close when clicking outside. Users must explicitly dismiss it using the close button or action buttons.

Use this variant for important confirmations or when you need to ensure the user acknowledges the content.
